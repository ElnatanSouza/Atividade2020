Atividade Computação em Nuvem Recode Pro 2020!

No curso Computação em Nuvem, foi solicitada a criação de uma VM (máquina virtual), virtualizada localmente, que deverá funcionar como um servidor de aplicação e de banco de dados. Nessa VM deve ser instalada a distribuição Linux Ubuntu Server. As atividades de instalação e configuração podem ser feitas diretamente no ambiente virtualizado, não sendo necessário o acesso via SSH.

Em Linux, pedimos para você demonstrar o seu conhecimento nos comandos Linux, acessando o ambiente virtualizado localmente, realizando a instalação dos pacotes necessários nos servidores (apache, aplicação e MySQL) e sua configuração.

Já em Servidores Web, sua missão foi a instalação de servidor HTTP, servidor de banco de dados e demais necessidades nas VMs (Apache, MySQL, PHP etc.).

Agora, em Banco de Dados em Nuvem, você deve criar um usuário para acesso remoto e pelo menos 1 (uma) view para acesso a dados de tabela.

Que a força esteja com você!
